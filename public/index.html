Important point

So now's the time to start building the CSS to be very actively targeting some or all of the various elements on th page(s).

Less important point

Allowing editors to create headers of multiple levels, makes it much harder to maintain the look and feel. Let alone consistency, which easily goes the way of the Dodo.

Subtlety

Worst case scenario, this should really be the smallest level of header allowed in a standard article.
