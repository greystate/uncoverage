Headers

Headers level 1–4 should cover most scenarios - Myself, I wouldn't ever create as many levels for anyone to use in the WYSIWYG editor of Umbraco.

Important point (level 1)

So now's the time to start building the CSS to be very actively targeting some or all of the various elements on the page(s).

Less important point (level 2)

Allowing editors to create headers of multiple levels, makes it much harder to maintain the look and feel. Let alone consistency, which easily goes the way of the Dodo.

Subtlety (level 3)

Worst case scenario, this should really be the smallest level of header allowed in a standard article.

Brevity for brevity's sake (level 4)

As it says - let’s try that.
